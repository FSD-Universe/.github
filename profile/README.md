致所有关注 FSD-Universe 项目的朋友：

我曾真心相信，开源能让我们的社区变得更好，也亲眼见证过它带来的积极改变。对于那些私下搭建服务器、与好友共同体验的玩家，我不仅不反对，反而感到欣慰——没有什么比看到自己写的代码真正帮到别人，更能让作者感到满足。

正因如此，我当初选择了 MIT 协议，希望以最大限度的宽松，降低使用门槛，让技术自由流动。

然而，过去一段时间里，这份信任被反复滥用。我的署名被从代码中移除，版权声明被删除，项目被当作“无主之物”随意分发、整合，甚至用于商业或未授权的再发布。这既违背了 MIT 协议的基本要求，也伤害了开发者最基本的尊重。

为此，我不得不作出一个艰难的决定：

自北京时间 2026 年 6 月 28 日 0 时 0 分起，

1. FSD-Universe 项目及其所属组织下的所有新版本代码，将不再以任何开源许可证发布，转为闭源；
2. 此前所有版本的源代码，也将不再对外开放；
3. 对于已发布且采用 MIT 协议的 Release 版本，其授权仍然有效，但仅限该特定版本，不延伸至后续任何更新。

同时，考虑到仍有部分飞友希望与朋友进行简单的联飞，我将推出一个功能受限的 Lite 版本，供有需要的用户免费使用。该 Lite 版本将采用更严格的许可证，以明确禁止商业使用、未经授权的再分发及任何形式的署名抹除。

特别提醒：
MIT 协议明确规定：“上述版权声明和本许可声明应包含在软件的所有副本或实质性部分中。”对于任何违反该条款的行为，本人保留向司法机关提起侵权诉讼及主张损害赔偿（含合理维权开支）的一切法律权利。

感谢真正理解并尊重开源的每一位朋友。我依然相信善意，但也必须用规则保护这份善意。
