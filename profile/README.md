# **FSD Universe**  
## **下一代飞行模拟联机基础设施**

FSD Universe 是一个专注于飞行模拟联机服务端及其生态系统的开源组织。

在传统飞行模拟联机领域，许多技术仍停留在十年前的水平。我们致力于通过现代化技术栈，推动联机服务向**高性能、高可用架构**演进，为整个生态注入新的活力。

---

## 🚀 核心项目

### **SimpleFSD — 现代化全功能 FSD 服务器**  
[![SimpleFSD](https://img.shields.io/github/v/release/FSD-Universe/SimpleFSD?label=SimpleFSD&logo=go)](https://github.com/FSD-Universe/SimpleFSD)

采用 Go 编写的高性能飞行模拟联机服务器，组织的旗舰实现。  

- **完整协议支持**：实现 FSD v3.000 Draft 9，兼容 VATSIM (TOKEN) 及 VATSIM2022  
- **专业级特性**：智能计划同步与锁定、增强管制员信息、高频率 VisualPosition 更新（支持 0.2 秒/次）  
- **多协议网关**：内置 TCP FSD、全功能 HTTP API 与 WebSocket，便于现代应用集成  
- **持续演进**：处于快速迭代期，为追求极致性能不断重构  

### **SimpleFSD‑Lite — 极简核心 FSD 服务器**  
[![Lite](https://img.shields.io/github/v/release/FSD-Universe/SimpleFSD-Lite?label=SimpleFSD-Lite&logo=go)](https://github.com/FSD-Universe/SimpleFSD-Lite)

### **SkyMesh — 分布式联飞解决方案（规划中）**  
下一代分布式微服务架构联飞方案，使用 Go 构建。

## 🧠 技术理念

- **性能优先**：利用 Go 原生高并发，支撑大规模联飞活动。  
- **模块化设计**：以微服务与模块化构建复杂而稳定的系统，未来逐步拆分独立服务。  
- **开放透明**：所有项目采用 **MIT 协议**，无附加条款，鼓励社区自由使用、修改与分发。  
- **实践驱动**：每个功能源于实际运营需求，确保切实有用。

---

## 🌱 生态与未来

我们正聚焦 **SimpleFSD** 核心，并在此基础上构建更丰富的生态：

- **微服务扩展**：规划独立服务，如专用语音中继、数据持久化与分析、AI 交通生成等。  
- **衍生工具**：基于 HTTP API 与 WebSocket，开发现代化 Web 雷达客户端等应用。
